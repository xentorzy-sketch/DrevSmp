## 📖 Über DrevSMP

DrevSMP ist ein exklusiver **Minecraft CitybBuild Server** mit einer professionellen Community und High-End-Verwaltung. Hier kannst du deine eigene Stadt bauen, mit anderen Spielern zusammenarbeiten und an einzigartigen Events teilnehmen.

### 🌐 Server Details
- **IP:** `Play.drevsmp.de`
- **Spielmodus:** CitybBuild

## 🚀 Wie man beitritt

1. **Discord-Server beitreten:** [discord.gg/frA4bwy8ph](https://discord.gg/frA4bwy8ph)
2. **Minecraft öffnen** und zum Multiplayer-Menü gehen
3. **Server hinzufügen:**
   - Servername: `DrevSMP`
   - Server-Adresse: `Play.drevsmp.de`
4. **Verbinden** und spielen! 🎉

## 📋 Wichtige Befehle

| Befehl | Beschreibung |
|--------|-------------|
| `/plot auto` | Weist dir ein freies Grundstück zu |
| `/plot home` | Teleportiert dich zu deinem Grundstück |
| `/rules` | Zeigt die Serverregeln |
| `/discord` | Öffnet den Discord-Link |
| `/help` | Zeigt alle verfügbaren Befehle |

## 📰 Regeln

- ✅ Respektiere andere Spieler
- ✅ Keine Griefing oder Diebstähle
- ✅ Keine Hacks oder Cheats
- ✅ Folge den Anweisungen der Moderatoren
- ✅ Bleibe aktiv auf deinem Grundstück (30 Tage)

Siehe [Rules Seite](https://xentorzy-sketch.github.io/DrevSmp/rules.html) für weitere Details.
